## Setup & Installation

1. Clone the repository:
```bash
git clone <repository-url>
```

2. Navigate to the project folder:
```bash
cd task6
```

3. Open `index.html` in your browser:
```bash
# On Windows
start index.html

# On Mac
open index.html

# On Linux
xdg-open index.html
```

That's it! No build process or dependencies required. Just open and run.
